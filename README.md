# Spatio-temporal Analysis of Food Availability in the USA

This repository contains the code used in the manuscript:

"Spatio-Temporal Trends of Food Availability: Implications for Food Insecurity and Waste in the United States"

The project develops a multi-scale analytical framework to study food availability (measured by food consumption expenditures) and its correspondence with food insecurity and residential food waste in the United States.

## Contents
- Colab notebook: main analysis
- Data sources: FRED, BEA, BLS, USDA ERS, EPA, ReFED

## Data Sources
All data used are publicly available:
- FRED (food expenditures, CPI, employment)
- USDA ERS (Food Availability Data System)
- BEA (state expenditures)
- BLS
- EPA
- ReFED

Raw datasets are not included in this repository.
